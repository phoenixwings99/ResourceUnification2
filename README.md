# Resource Unification 2

## What This Does 
Resources that should be the same (ki, arcane pool, arcane resivour, bombs) across classes are now the same in game by creative post-fixing to send all requests for the resource to the right place and patching to look to the right stat for max values.
No blueprints are created.

As an optional component, it removes some of archetype changes that are only needed to support resources with alternate stats. Further work on this will continue. (Thiefling will happen as soon as I get Woljif in my current playthrough.)

If you encounter a multi-classing where you think this should apply, or an ability using those resources that isn't working right, open an issue.

Mod interoperability will be provided, just open an issue.
